# Administrator Guide

This guide covers everything a School Administrator needs to know to manage a successful event using Taqeem.

## 1. Managing Competitions
The heart of Taqeem is the **Competition**.
- **Creating a Competition**: Go to the Admin panel and select "Competitions." Give your event a clear name (e.g., "Annual Science Fair 2024").
- **Switching Context**: You can switch between competitions using the "Switch Competition" button. This updates your entire dashboard to show data only for that specific event.

## 2. Importing the Roster
Don't add students one by one!
- Prepare a CSV file with columns like `name` and `student_number`.
- Use the **Import Roster** tool in the Admin sub-tab.
- The system will automatically link these students to your active competition.

## 3. Managing Judges
- **User Accounts**: Create unique usernames and passwords for your judges.
- **Assignment**: When creating a judge account, you MUST assign them to a specific competition. They will only see the students and evaluations for that assigned event.
- **Global Admins**: Administrators have access to all competitions and can manage the entire system.

## 4. Monitoring Progress
- As judges perform evaluations, you can view the "Evaluations" tab to see real-time scores.
- Use this to ensure all students are evaluated before the final announcement.

## 5. Announcing Winners
- Once all scores are in, navigate to the **Winners** screen.
- This screen automatically calculates rankings and presents the top performers in a beautiful layout perfect for projecting on a stage.
